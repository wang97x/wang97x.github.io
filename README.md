## Writing Posts

This repository includes a reusable workflow for turning study notes into `_posts` articles.

- Writing guide: [tools/blog-writing-workflow.md](/E:/Blog/vaxh.github.io/tools/blog-writing-workflow.md)
- Post scaffold script: `powershell -ExecutionPolicy Bypass -File .\tools\new-post.ps1 -Title "文章标题"`
- Interactive helper: `.\tools\MDCreating.bat`
